Portfolio

SMS-based Service Locator
Tools used:
  • Django
  • RapidSMS

This tool is designed to allow people living in temporary settlements to locate resources and services near them. To compensate for how inconvenient it is for SMS messages to carry positional data, it relies on the concept of a 'beacon'—a point from which directions may be calculated.

Planned functionality: graphic reports on query metrics, scheduled monitoring and threshold-based alerts.

Order Fulfillment Inventory Tracker
Tools used:
  • Django
  • PostgreSQL

This tool is intended for tracking accounts and inventory for customers of an order fulfillment business. It supports tracking accounts, shipments containing cartons, status change operations, and costs for billing purposes.

Planned functionality: generate PDF invoices, API-based connection to front-end site.
