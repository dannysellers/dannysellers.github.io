SMS-based Service Locator
Tools & Stack:
  • Django
  • RapidSMS
  • PostgreSQL
  • jQuery
  • Bootstrap

This tool is designed to allow people living in temporary settlements to locate resources and services near them. To compensate for the inconvenience of transmitting positional data over SMS, this project relies on the concept of a 'beacon'—a point from which directions are calculated.

Planned functionality: graphic reports on query metrics, scheduled monitoring and threshold-based alerts.
